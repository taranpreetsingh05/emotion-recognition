Problem Statement

Most emotion recognition systems analyse utterances in isolation, ignoring how emotions evolve over a conversation. This project builds a context-aware AI system that maintains temporal consistency across conversational turns.

Our Approach

We compare a Baseline TF-IDF + Logistic Regression model against a Context-Aware version that uses a sliding-window history buffer and a transition cost matrix to smooth abrupt emotion shifts.

Methodology

1. Dataset collection (MELD-aligned synthetic data)
2. Text preprocessing & feature engineering
3. Baseline model development
4. Context-aware extension with temporal smoothing
5. Emotion flow analysis
6. Performance evaluation (Accuracy, Precision, Recall, F1)
